# Radix Sorting Project

### Description

This project demonstrates the use of queues in the execution of a radix sort on a set of numeric values in an arrayList.

### To Run

Using terminal on Mac:

```
$ cd Radix-Sorting
$ javac RadixSort.java
$ java RadixSort
```
